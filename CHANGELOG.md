# Changelog

All notable changes to laneq are documented here. Versions follow
[semantic versioning](https://semver.org/) and are published as git tags
(`vX.Y.Z`). Entries are derived from the merged history at each tag.

## [v0.4.0] - 2026-06-17

### Added
- laneq MCP server, exposing the queue as MCP tools (#6).
- Code-style CI gates.

## [v0.3.0] - 2026-06-13

### Changed
- Rename the package and CLI from `codex-q` to `laneq`.

## [v0.2.0] - 2026-06-13

### Added
- v2 coordination: lease-based, multi-consumer queue semantics.

## [v0.1.0] - 2026-06-13

### Added
- Initial release: the priority-queue CLI (originally packaged as `codex-q`) (#1).
